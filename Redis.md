## Data types

- Lists of strings
- Sets of strings
- Sorted sets of strings
- Hash tables
- HyperLogLogs
- Stream of entries with consumer groups
- Geospatial data
- JSON (RedisJSON)
- Graph (RedisGraph)
- Time series (RedisTimeSeries)
- Bloom filter, Cuckoo filter, Count-min sketch and Top-K (RedisBloom)
